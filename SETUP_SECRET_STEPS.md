# ⚡ Quick Setup - 2 Minutes

## Step 1: Add GitHub Secret

1. **Open this link in your browser:**
   ```
   https://github.com/llomj/SophiAI/settings/secrets/actions
   ```

2. **If not signed in, sign in to GitHub**

3. **Click the green button:** "New repository secret" (top right)

4. **Fill in:**
   - **Name field:** Type: `GEMINI_API_KEY`
   - **Secret field:** Type: `AIzaSyCDaBAfP2zefxS4x5JkEu6UbS5Koq3N4HE`

5. **Click:** "Add secret" (green button at bottom)

## Step 2: Trigger Build

1. **Go to:** https://github.com/llomj/SophiAI/actions

2. **Click on:** "Deploy SophiAI" workflow

3. **Click:** "Run workflow" button (top right, green button)

4. **Select branch:** Choose "main" from dropdown

5. **Click:** "Run workflow" (green button)

6. **Wait 2-3 minutes** for the build to complete

## Step 3: Update Phone App

After the build completes:

1. **Remove the app** from your phone's home screen
2. **Clear browser cache** on your phone
3. **Visit:** https://llomj.github.io/SophiAI/
4. **Re-add to home screen**

✅ Done! Your phone app will now work with the API key.

**Need help?** The build will show ✅ when it's done. Check the Actions page to see progress.
